Workshops - Get comfortable with Data Visualization!

Want to improve your data visualizations? The first day (May 10) of this workshop will cover how to choose the right chart type for your data and how good design choices will make your chart easier to understand. The focus will be on visualization best practices, independent of any specific visualization software.

The rest of the workshop (May 11-12) is an introduction to making interactive data visualizations in Python. Learners will wrangle data into the proper format using the pandas library, create visualizations using the Plotly Python library, and display these visualizations and create widgets using Streamlit .

Workshops will be held online via Zoom.
