Print from Personal Windows Computer / Remap a Printer

If you need more in-depth assistance on locating proper drivers or connecting to the printer, please contact library@caltech.edu or call x6775.

  1. Bring up the Run window by pressing the Windows key and R key
  2. Type the following in the Open: \\131.215.225.26 (if you're Library Staff you can alternatively utilize \\printers) and then click the OK button (or press the Enter key):
  3. At this point you will be prompted for your access.caltech credentials. In the username field enter your access.caltech username and make sure you preface it with ITS\. An example would look like: ITS\username. Your appropriate password should be utilized in the password field. Check the Remember my Credentials checkbox. Click the OK button:
  4. A new window should open up showing you a list of all the available printers. There are only four (4) printers that you can authenticate against with your access.caltech credentials. All other printers will fail. The following four (4) printers are available to you no matter which library branch you are printing to:
    1. library-color-duplex - This queue defaults to duplex (double-sided) Color.
    2. library-color-simplex - This queue defaults to simplex (singled sided) Color.
    3. library-grayscale-duplex - This queue defaults to duplex (double-sided) Grayscale (Black & White).
    4. library-grayscale-simplex - This queue defaults to simplex (singled sided) Grayscale (Black & White).
  5. Double-Click any, or all, of the previously mentioned printers in step 7. (You can also right click the appropriate printer icon and then select Connect… from the menu.) There might be a slight delay while the printer server ensures you have the proper printer drivers for your selection:
  6. At this point it might take several seconds but a window will quickly show as drivers are being installed.
  7. Once the drivers are successfully installed, you should see the printers available in your Settings menu.

Print from Personal Apple Computer

If you need more in-depth assistance on locating proper drivers or connecting to the printer, please contact helpdesk@library.caltech.edu or call x6775.

Part 1: Installing the Printer Driver

  1. Identify the make and model of the printer you wish to connect to. In this case we will be attempting to connect to a Xerox C8035 printer. This information is important because you might need to install drivers onto your Apple computer.
  2. The Caltech Library does not host drivers for download. You will need to go to the appropriate vendor website and acquire the proper drivers based on make/model and operating system. In this example you will need drivers for either the Xerox C8035 or the Xerox C8055.
  3. Go to: https://www.support.xerox.com/support/altalink-c8000-series/downloads/enus.html?associatedProduct=C8000-series-built-in-controller&operatingSystem=macosx10_14&fileLanguage=en
  4. Under Drivers & Downloads: under Platform, select the operating system on your computer; under Category, select Drivers; and under Language, the language you wish to use. Select Apply Filters.
  5. Note the Filename for the Driver file. Select the box next to "I agree to the Terms and Conditions", then select DOWNLOAD to download the driver .dmg file. This should create a virtual drive on your Mac Desktop. It should automatically open after downloading.
  6. Within the virtual drive, locate the .pkg file and double-click to start the installation process.
  7. If message "This package will run a program to determine if the software can be installed" displays, click Allow.
  8. At "Introduction" click Continue.
  9. At "License" click Continue.
  10. Click "Agree" to agree to the terms of the license.
  11. At "Installation Type" click Install.
  12. If prompted, enter your local machine login password (note this is NOT your access.caltech password - this is what you use to log in to your machine.)
  13. At "Select A Printer", do not select a printer. Click Continue.
  14. At the message "Print Queue Not Created" select Continue.
  15. At "Download App", you can choose to download the app but it is not necessary. Click Continue.
  16. You should see the message "The installation was successful. The software was installed."
  17. Continue to Part 2.

Part 2: Configuring the CUPS Printer Server

  1. Launch a web browser and in the URL bar enter the following: localhost:631 for the address.
  2. If you get a message "The web interface is currently disabled" or another error message, launch the Terminal application and enter the following: cupsctl WebInterface=yes (and press enter). When complete, exit Terminal. Reload the browser window with localhost:631.
  3. You should now be on the main CUPS page. The version of CUPS may vary - the latest available version is 2.3.6 (see cups.org ).
  4. Click "Administration" in the top menu. Then, under the "Printers" section, select "Add Printer button".
  5. If prompted, enter your local machine login name and password (note this is NOT your access.caltech credentials - this is what you use to log in to your machine.)
  6. Under "Add Printer", select the LPD/LPR Host or Printer bullet and click Continue.
  7. There are four Library print queues available:
    1. library-color-duplex – This queue defaults to duplex (double-sided) Color.
    2. library-color-simplex – This queue defaults to simplex (single sided) Color.
    3. library-grayscale-duplex – This queue defaults to duplex (double-sided) Grayscale (Black& White).
    4. library-grayscale-simplex – This queue defaults to simplex (single sided) Grayscale (Black & White).
  8. In the "Connection" box, enter one of the following four printer choices:
    1. lpd://access.caltech.username@131.215.225.26/library-color-duplex
    2. lpd://access.caltech.username@131.215.225.26/library-color-simplex
    3. lpd://access.caltech.username@131.215.225.26/library-grayscale-duplex
    4. lpd://access.caltech.username@131.215.225.26/library-grayscale-simplex
  9. IMPORTANT: It is very important that you replace the access.caltech.username variable with your correct access.caltech username. This is how you authenticate to the printers. For example, if your access.caltech username is mauricemoss, then you would enter: lpd://mauricemoss@131.215.225.26/library-color-duplex. The access.caltech username must be correct otherwise the card release will not recognize the print job. After entering the appropriate information, select Continue.
  10. On the next "Add Printer" page, you may enter any custom data you like for the Name, Description, and Location. It will not affect print functionality. Select Continue.
  11. On the next "Add Printer" page, under the "Make" pulldown menu, select Xerox and select Continue.
  12. On the next "Add Printer" page, scroll down until you locate the appropriate Printer model/driver. Once it is selected, select Add Printer.
  13. If the next page displayed is "Set Printer Options", then the printer has successfully been added.
  14. You can repeat these Part 2 steps to add other print queues as needed.
