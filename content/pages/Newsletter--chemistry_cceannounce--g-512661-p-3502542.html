SciWheel Institutional Subscription Cancelled

Caltech Library has cancelled its subscription to SciWheel. This citation management tool was previously included with other library subscriptions from F1000 but has recently been purchased by the publisher SAGE. The institution's SciWheel subscription ended on November 30, 2023. Those using SciWheel can either switch to the Library-supported citation manager Zotero ( more information here ) or switch to a personal SciWheel account. Please contact library@caltech.edu with any questions.

F1000 Prime / Faculty Opinions Access Changing

H1 Connect, formerly F1000 Prime and Faculty Opinions, is discontinuing institutional subscriptions as of January 31, 2024. After January 31th, 2024 you can maintain free access to the H1 Connect content without disruption by creating a free account.

Authentication to Library Resources Using OpenAthens

Caltech Library has changed its off-campus authentication service from EZProxy to OpenAthens. OpenAthens is an authentication service that allows you to access Library-subscribed electronic resources such as ejournals, ebooks, and online databases from off-campus, using your access.caltech username and password. If you have resources bookmarked with the old EZproxy ( caltech.idm.oclc.org ), those links will no longer work . You will need to access resources through the new OpenAthens system and bookmark new links.

If you are on campus or if you are using the Caltech VPN , you do not need to authenticate with OpenAthens to access e-resources. OpenAthens is for off-campus use. (Please note that if you use VPN, be sure and use Option 2, the Tunnel-all option, and note that this may slow your Internet connection).

If you find an item you’d like to access, and you see the Get It @ Caltech button, select that. This will check if we have access to that electronic resource. If we do have access, you may be asked to authenticate in using your access.caltech credentials. If we do not, you have the option to use Docuserve to request the item from another library.

Another option you can use is the OpenAthens prefix: https://go.openathens.net/redirector/caltech.edu?url=

Enter the complete web address after the = sign, and then select "Caltech Library" at the next screen. Then enter your access.caltech credentials. Please note that if you sign in, it does not necessarily mean we have paid access to that subscription. If we do not have access, please use Docuserve .

If you are off campus, using VPN is preferred (although your Internet connection will slow down), as you do not need to use any authentication measure. If you come across a resource that you cannot get access to, it means the library does not have paid subscription to that particular resource – please use Docuserve to request that material.
