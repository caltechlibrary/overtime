Accessing Full Text

From the main Search Results screen, click on the "Get It @ Caltech" icon to check for access from the Caltech Library:


From an individual Web of Science record for  an article, it can be found in the upper left:

Clicking on Get It @ Caltech will check our catalog to see if we have electronic access to the article. There may be multiple links, if we have access through multiple sources - be sure to choose the link that corresponds with the year or volume range that you need.

Sometimes, particularly for older (pre-1997) years of journals, the Library only has print copies available.

If the Library doesn't have any subscription to the journal, you can request a copy of the article via DocuServe, using the Request delivery via DocuServe link under Document Delivery & ILL . There is no charge to request copies of articles to which Caltech does not have access. (More information about DocuServe)
