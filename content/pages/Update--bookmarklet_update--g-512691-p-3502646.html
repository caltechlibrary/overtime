EZproxy switching to OpenAthens

The Library is switching off-campus authentication methods from EZproxy to OpenAthens.

Please update any bookmarked links before Wednesday, August 16, 2023.

What you need to know:

  • If you have any bookmarked links that start with:
    • https://clsproxy.library. caltech.edu/login?url= OR https://caltech.idm.oclc.org/ login?url=
    • Please replace them with https://go.openathens. net/redirector/caltech.edu? url=
  • You can also update links by using the OpenAthens link generator .
  • See the FAQ " How do I update my proxied bookmarked links to work with OpenAthens? "
  • There is no replacement for the proxy bookmarklet. The bookmarklet will not work after August 16.
  • If you use Caltech's VPN to access library resources from off campus, you will be able to access resources without signing in to OpenAthens. Read more about Caltech's VPN .
  • Send link issues and questions to library@caltech.edu .
