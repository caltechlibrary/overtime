Access an EBSCO ebook

1. Search the library catalog . (Example title: "The data librarian's handbook")

2. Expand "Full Text Access," then select EBSCO eBooks or EBSCOhostEbooks

NOTE: If off-campus or using VPN, you may need to enter your Caltech credentials on the following proxy login page to proceed.

The Ebsco Ebook record page will open in a new tab.

NOTE: If off-campus or using VPN, you may need to enter your Caltech credentials on the following proxy login page to proceed.

3. Select one of the following options to access the ebook (do not select the Get It @ Caltech button):

a. ​ PDF Full Text : Select to read online and download chapters. The content and book chapters will be displayed on the left panel. Note the limits of PDF pages you can download. (Your session may timeout after browser inactivity).

b. Full Download : Select to download the whole ebook for 7 days. This option requires an EBSCO account and Adobe Digital Editions. Instructions are below in the next box.

NOTE: Some ebooks have a limit on the number of people who can use them at the same time (concurrent user limit). If the ebook has reached its limit, you will see this notice: “Sorry, this eBook is in use.” Check back periodically to use the ebook when it becomes available. Look at the bottom of the record for permissions and concurrent user level.

Full Download (from step 3b above)

1. Select Full Download. You MAY be asked to sign-in or create an Ebsco account if you do not have one. Otherwise, you can skip the next two steps if it does not ask you to sign-in/create account.

2. On the Personal Data Retention and Usage window, select Yes to finish creating an account to read the book. Select No to return to the record and access the ebook another way.

3. Select PDF (for desktop) or EPUB (for mobile devices)  If you haven’t before, install Adobe Digital Editions , then check the Adobe box and select the Full Download button. Select the number of days you'd like to borrow it for (1 to 7 days).

4. Open the downloaded ebook from your file system or desktop - it will have the file extension “.ascm”.

5. Complete authorization (if you haven't before): Check “I want to authorize my computer without an ID”, then select Next, then Authorize, then OK.

6. Success! You should now be able to read the ebook in Adobe Digital Editions. Need help? Contact the library .

NOTE: To view the due date or return the ebook, select Library in the upper left-hand corner, right-click the ebook, then select “Return borrowed item.”
